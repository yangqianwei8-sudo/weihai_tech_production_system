维海科技生产信息化管理系统

API 文档

版本: 1.0.0 | 最后更新: 2025-11-06

系统管理 API

GET /api/system/users/
获取用户列表，需要认证
POST /api/system/users/
创建新用户，需要管理员权限
GET /api/system/departments/
获取部门列表
POST /api/system/users/login/
用户登录

项目中心 API

GET /api/project/projects/
获取项目列表
POST /api/project/projects/
创建新项目
GET /api/project/projects/{id}/
获取项目详情
GET /api/project/projects/dashboard/
获取项目中心仪表盘数据

客户管理 API

GET /api/customer/clients/
获取客户列表
POST /api/customer/clients/
创建新客户
GET /api/customer/clients/statistics/
获取客户统计信息

其他端点

GET /api/
API 根目录，显示所有可用端点
GET /health/
系统健康检查

使用说明

  • 大多数 API 端点需要认证后才能访问
  • 请先通过 /api/system/users/login/ 登录获取会话
  • 或使用管理员账号登录管理后台：/admin/
  • API 返回格式为 JSON
