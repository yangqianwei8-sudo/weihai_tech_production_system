create_form_base.html 模板说明

模板概述

这是一个共享的表单创建模板，提供了统一的表单结构和布局规范。

布局规则
  • 每排显示5个字段：使用 form-row-5 类和 col-5-fields 类
  • 多行文本字段：使用 col-12，占据整行
  • 多选字段：使用 col-12，占据整行
  • 字段包装：每个字段必须使用 form-field-wrapper 类包装

模板结构
{% verbatim %}{% extends "shared/create_form_base.html" %}

{% block form_title %}创建XXX{% endblock %}
{% block form_subtitle %}请填写表单信息{% endblock %}

{% block form_first_row_fields %}
  {# 第一排字段（最多5个） #}
  <div class="col-5-fields form-field-wrapper">
    {# 字段内容 #}
  </div>
{% endblock %}

{% block form_fields %}
  {# 其他字段 #}
{% endblock %}

{% block form_actions %}
  <button type="button" class="btn btn-primary" id="headerSubmitButton">创建</button>
  <a href="{% url 'xxx:list' %}" class="btn btn-secondary">取消</a>
{% endblock %}{% endverbatim %}
可用的块（Blocks）
块名称 说明
form_title 表单标题
form_subtitle 表单副标题
form_first_row_fields 第一排字段（最多5个字段）
form_second_row_fields 第二排字段（可选）
form_fields 其他表单字段
form_actions 表单操作按钮（创建、取消等）
form_header_actions 顶部操作按钮（返回列表等）
form_approval_workflow_card 审批流程配置卡片（可选）
注意事项
  • 创建按钮必须使用 id="headerSubmitButton"，JavaScript 会监听此按钮的点击事件
  • 如果字段不足5个，使用空的 <div class="col-5-fields form-field-wrapper"></div> 占位
  • 多行文本字段和多选字段必须使用 col-12，不使用 form-row-5
{% endblock %}
