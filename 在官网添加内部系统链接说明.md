# 在维海科技官网添加内部系统链接说明

## 目标
在 https://www.vihgroup.com.cn/ 的导航菜单中添加链接，指向内部系统：https://rasdmangrhdn.sealosbja.site

## 方案一：通过WordPress后台管理添加（推荐）

### 步骤：

1. **登录WordPress后台**
   - 访问：`https://www.vihgroup.com.cn/wp-admin/`
   - 使用管理员账号登录

2. **进入菜单管理**
   - 在左侧菜单栏找到 **外观** → **菜单**
   - 或者直接访问：`https://www.vihgroup.com.cn/wp-admin/nav-menus.php`

3. **添加自定义链接**
   - 在左侧找到 **自定义链接** 部分
   - 填写以下信息：
     - **URL**: `https://rasdmangrhdn.sealosbja.site`
     - **链接文字**: `内部系统` 或 `管理系统`（根据您的需求）
   - 点击 **添加到菜单**

4. **调整菜单位置**
   - 在右侧菜单结构中，拖动新添加的菜单项到合适的位置
   - 建议放在"联络"之后，或者您认为合适的位置

5. **保存菜单**
   - 点击右上角的 **保存菜单** 按钮

## 方案二：直接修改主题文件（需要FTP/文件访问权限）

如果您有主题文件的访问权限，可以直接修改导航菜单的代码。

### 步骤：

1. **找到主题文件**
   - 通常位于：`wp-content/themes/[主题名称]/header.php` 或相关模板文件
   - 或者通过WordPress后台：**外观** → **主题编辑器**

2. **定位导航菜单代码**
   找到类似这样的代码：
   ```html
   <ul id="main-nav" ...>
       <li class="menu-item ..."><a href="..."><span>首页</span></a></li>
       <li class="menu-item ..."><a href="..."><span>项目</span></a></li>
       <li class="menu-item ..."><a href="..."><span>动态</span></a></li>
       <li class="menu-item ..."><a href="..."><span>关于</span></a></li>
       <li class="menu-item ..."><a href="..."><span>联络</span></a></li>
   </ul>
   ```

3. **添加新菜单项**
   在最后一个 `<li>` 标签后添加：
   ```html
   <li class="menu-item menu-item-type-custom menu-item-object-custom">
       <a href="https://rasdmangrhdn.sealosbja.site" target="_blank">
           <span>内部系统</span>
       </a>
   </li>
   ```

4. **保存文件**

## 方案三：通过functions.php添加（高级）

如果主题支持，可以在 `functions.php` 中添加代码来自动添加菜单项：

```php
// 在functions.php中添加以下代码
function add_internal_system_menu_item($items, $args) {
    if ($args->theme_location == 'primary') { // 根据您的菜单位置调整
        $items .= '<li class="menu-item menu-item-type-custom menu-item-object-custom">
            <a href="https://rasdmangrhdn.sealosbja.site" target="_blank">
                <span>内部系统</span>
            </a>
        </li>';
    }
    return $items;
}
add_filter('wp_nav_menu_items', 'add_internal_system_menu_item', 10, 2);
```

## 注意事项

1. **备份**：在修改任何文件之前，请先备份
2. **子主题**：如果使用子主题，请在子主题中修改，避免主题更新时丢失修改
3. **权限**：确保有足够的权限访问WordPress后台或文件系统
4. **测试**：添加后请测试链接是否正常工作

## 推荐方案

**推荐使用方案一（WordPress后台管理）**，因为：
- 不需要修改代码
- 更安全，不会因为主题更新而丢失
- 更容易管理和维护
- 可以随时修改或删除

## 如果无法访问后台

如果您无法访问WordPress后台，请联系：
- 网站管理员
- 网站开发人员
- 或者提供FTP/服务器访问权限，以便我们帮您修改
